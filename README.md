🔐 Linux Hardening Toolkit (CentOS)

📌 Overview

Automated Linux server hardening for CentOS/RHEL systems using Bash scripting.

🚀 Features

- SSH security (disable root + password login)
- Firewall configuration (firewalld)
- Fail2Ban protection
- System updates

🛠️ Tech Stack

- CentOS / RHEL
- Bash scripting

▶️ Usage

chmod +x scripts/harden.sh
./scripts/harden.sh
